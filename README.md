Uygulama İşlevi

Windows formlar üzeriden sql ile temel veri tabanı sorgularını yürütür (kayıt ekleme, kayıt listeleme, tablo birleştirme, log kaydı tutma).

Uygulamanın Çalıştırılması :

Uygulama microsoft visual studio aracılığıyla c# yazılım dili ve windows forms app (net. framework) kitaplığı ile programlandı, microsoft sql server ve sql server management studio ile veri tabanı oluşturuldu.

a) Microsoft Visual Studio, Microsoft SQL Server, SQL Server Management Studio programlarını yükle.

b) SQL Server Management Studio da “Proje”klasörde bulunan “veritabani.bacpac” dosyasını import et ve veri tabanına erişim sağla.

c) Microsoft Visual Studio da “Proje” klasöründe bulunan “vtys.sln” dosyayı aç ve çalıştır.
